# My Portfolio

This is a simple static personal portfolio site. Replace content, images, and links with your own.

## Files

- `index.html` — main page
- `styles.css` — styling
- `script.js` — minimal JS for nav + year
- `assets/` — images, resume

## Deploy with GitHub Pages

1. Create a new GitHub repository (public).
2. Push this project to the repository.
3. In repository settings → Pages, select branch `main` and folder `/ (root)` (or `gh-pages` if you prefer).
4. The site will be published at `https://<your-username>.github.io/<repo>/` within a minute.

Alternatively, move files into `docs/` and select `docs/` in Pages settings.
